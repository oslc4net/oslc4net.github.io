Class ServiceProviderRegistryClient

Namespace
OSLC4Net.Client
Assembly
OSLC4Net.Client.dll

This class provides methods to register and deregister with an OSLC ServiceProvider Registry (not yet implemented in OSLC4Net - see Eclipse Lyo for a Java implementation)

It also provides methods to get a ServiceProviderCatalog and retrieve the ServiceProviders
